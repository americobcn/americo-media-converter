Data flow

startConversion() → switch tab label "Normalize" → convertNormalize()
    → for each file: cv.normalize(file, targetLUFS, scriptURL, jqURL, row)
    → Process: /bin/bash normalize_r128.sh -i <LUFS> <inputFile>
    → stdout parsed: Pass 1/2 → 30% · Pass 2/2 → 70% · [OK] → 100%
    → delegate callbacks: conversionProgress(row, percent) + shouldUpdateOutView(line)

Script invocation

// Process setup in normalize()
executable: /bin/bash
arguments: [scriptPath, "-i", "\(targetLUFS).0", inputFile.path]
environment: PATH = Resources/ + /usr/bin:/bin:/usr/local/bin
// jq found via PATH pointing to bundled Resources/jq
// Output file handled by script: same dir, adds suffix _EBU_R128(-23LUFS)
